# cora1022

백엔드 구조, 자동화 흐름, 데이터 처리 과정을 직접 만들고 운영하며 기록합니다.

[개인 포트폴리오](https://cora1022.com/) · [개발기록](https://cora1022.com/blog/) · [이미지변환소](https://image.cora1022.com/)

## About

- Java/Spring Boot와 Python/FastAPI를 중심으로 백엔드 구조를 공부하고 있습니다.
- 알고리즘 문제 풀이를 꾸준히 진행해 과거 백준 **Gold V**를 달성했습니다.
- **NYPC 마스터트랙 NEXT NATION**에 참가해 전략 봇을 반복 개선하고, 제출 과정과 시행착오를 기록했습니다.
- 기능 구현뿐 아니라 배포, 로그 확인, 문서화까지 하나의 개발 과정으로 다루는 것을 중요하게 생각합니다.

## Tech

- **Backend:** Java, Spring Boot, Python, FastAPI
- **Data:** MySQL, Qdrant
- **Frontend:** JavaScript, TypeScript, React, Astro
- **Vision:** OpenCV, OpenCLIP
- **Tools:** Git, GitHub, Docker, Netlify, GitHub Pages
- **Hardware:** Raspberry Pi

## Selected Work

### 이미지변환소

PNG, JPG, WebP 형식 변환과 압축, 크기 변경, EXIF 제거 기능을 제공하는 웹 도구입니다. Astro와 TypeScript로 구성하고 Netlify 자동 배포로 운영합니다.

- [서비스](https://image.cora1022.com/)

### OpenCV 기반 패션 이미지 유사도 검색

OpenCV/YOLO로 의류 영역을 전처리하고, OpenCLIP 임베딩과 Qdrant 벡터 검색을 연결한 이미지 검색 프로젝트입니다.

- [Repository](https://github.com/cora1022/OpenCV-pj-Fashion-Model)

### ROBOT

Raspberry Pi 로봇을 자연어 명령으로 제어하는 Python 프로젝트입니다. 센서 입력, Gemini function calling, 이동·정지·회전·음성 응답 흐름을 나누어 실험했습니다.

- [Repository](https://github.com/cora1022/robot)

### 도시 폐기물 통계 시뮬레이션

건물 유형과 인구 배율을 조절해 도시 폐기물 데이터를 합성하고 CSV로 내보내는 정적 웹 시뮬레이션입니다.

- [Demo](https://cora1022.com/Urban-Waste-Statistics-Simulation/)
- [Repository](https://github.com/cora1022/Urban-Waste-Statistics-Simulation)

## Experience & Records

NYPC 마스터트랙 NEXT NATION 예선에서 전략 봇을 설계하고 제출했습니다. 리플레이와 중간 평가 결과를 바탕으로 경로 예측, 방어 우선순위, 본부 업그레이드 로직을 개선했습니다.

- [NYPC 참가 타임라인](https://cora1022.com/blog/posts/nypc-participation-timeline.html)
